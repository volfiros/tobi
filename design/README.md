# Tobi Dashboard Design

Stitch project: `projects/4694781719257843021`

The dashboard uses Stitch as the design source for the visual direction:

- light, operational interface for print shop staff
- dense order tables instead of marketing cards
- deep green primary action color
- neutral surfaces with amber, blue, green, and red status colors
- compact status controls that work on desktop and mobile

Generated images, screenshots, and scratch exports should stay under ignored
paths such as `design/generated/`, `design/screenshots/`, or
`.agent-artifacts/stitch-notes/`.
